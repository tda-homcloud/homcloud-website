周期アルファフィルタレーション¶

対象となるデータはperiodic_pointcloud.txtというファイルです。 これは3次元一様分布に従うランダムな点(各座標0~1)を5000個撒き散らしたものです。これを解析してみましょう。

このようにデータが3列で並んでいます。 データから周期的パーシステント図を計算します。 --periodicというオプションを使用します。

In [2]:
python3 -m homcloud.pc_alpha -d 3 --periodic 0 1 0 1 0 1 periodic_pointcloud.txt periodic_pointcloud.pdgm

0 1 0 1 0 1はそれぞれデータのx座標の最小値、最大値、y座標の最小値、最大値、z座標の最小値、最大値を表しています。 一様分布のデータを使用する理由は、周期境界条件を付けるわけなので境界の端の方にもそれなりに点が詰まっていないと上手く動作しないからです。 0 1 0 1 0 1にした場合、{0,1}x{0,1}x{0,1}を一単位の周期ボックスとしてそれをプログラム側が並べて計算します。周期的なドロネー三角形分割を作るのですが周期ボックスの端っこの方でそれを張り合わせて周期的な三角形分割を作るようになっています。なので周期ボックスの端っこの方にそれなりに点が詰まっていないと、同じ始点と終点を共有する辺が出来たり、ループ辺ができたりするので、数学理論とプログラムの実装の両方の問題で都合が悪いです。

パーシステント図の可視化¶

次にここからパーシステント図をプロットしてみましょう。

In [3]:
python3 -m homcloud.plot_PD -d 1 periodic_pointcloud.pdgm -o periodic_pointcloud-pd1.png

すると以下のような画像が出力されます。

In [4]:
display < periodic_pointcloud-pd1.png










    




























スケールをログスケールに直します。













In [5]:



    


python3 -m homcloud.plot_PD -d 1 -l --aspect equal periodic_pointcloud.pdgm -o periodic_pointcloud-pd1-log.png
display < periodic_pointcloud-pd1-log.png

簡単な逆解析¶

In [11]:



    


python3 -m homcloud.dump_diagram -S yes -d 1 periodic_pointcloud.pdgm -o periodic_pointcloud-pd1.txt
# head で最初の3行だけ表示している
head -n 3 periodic_pointcloud-pd1.txt

高度な逆解析機能(Optimal Volume)¶
次は HomCloud の強力な逆解析ツールである、optimal volumeを使いましょう。
とりあえずさきほど表示したパーシステント図をもう一度見てみましょう。













In [12]:



    


display < periodic_pointcloud-pd1-log.png



    















    















    




























(0.0008, 0.0032) 付近に birth-death pair があります。これは
対角線から離れているため、他のリングに比べて重要度が高そうです。これ
の optimal volume を調べましょう。homcloud.optimal_volume モジュールを使います。
以下のようなコマンドを実行します。













In [14]:



    


python3 -m homcloud.optvol -d 1 -x 0.0008 -y 0.0032 -P periodic_pointcloud.pdgm



    















するとparaviewのウィンウドウが表示されたはずです。周期境界条件の計算は結構高コストです．
左側のパネルの「Apply」ボタンを押すと表示がされます。




表示は周期境界条件を考慮しないので可視化はけっこうぐちゃぐちゃになります．これは今後改善する予定です．


この情報は
ファイルに保存することもできます。以下のようにします。













In [15]:



    


python3 -m homcloud.optvol -d 1 -x 0.0008 -y 0.0032 -j periodic_optimal_volume.json periodic_pointcloud.pdgm



    















periodic_optimal_volume.jsonにoptimal volumeの情報が保存されています。
