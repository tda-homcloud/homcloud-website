# HomCloud

HomCloud はパーシステントホモロジー(persistent homology)にもとづいた
データ解析ソフトウェアです。
[大林](http://www.wpi-aimr.tohoku.ac.jp/hiraoka_labo/obayashi/)を中心に
[平岡研](http://www.wpi-aimr.tohoku.ac.jp/hiraoka_labo/)で開発されています。
